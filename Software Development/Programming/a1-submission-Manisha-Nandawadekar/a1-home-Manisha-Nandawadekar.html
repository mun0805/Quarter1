"What is HTML?"

  • HTML stands for Hypertext Markup Langauge.
  • Using HTML you can create website.
  • It has series of element that instruct browser How to display content
  • All elements represented in tag
  • HTML can be used along with technologies like CSS and scripting languages like javascript to provide creative web applications
  • It is Developed by Web Hypertext Application Technology Working Group (WHATWG)
  • The most recent version of HTML is 5.0
