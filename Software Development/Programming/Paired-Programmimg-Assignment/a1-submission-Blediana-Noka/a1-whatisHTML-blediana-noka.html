HTML is the acronym for Hypertext Markup Language.

HTML tells Web browsers how Web pages should look.

Various languages such as Java, Flash and Javascript can work with HTML.

Different tools can be used to create HTML files such as Sublime Text, Visual Studio Code and Notepad++.

Cascading Style Sheets (CSS) assists by creating styles in a HTML file.

HTML docs can be saved in a text format and is possible to view it or edit it on any operating system that is connected to the internet.
